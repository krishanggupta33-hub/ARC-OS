It is a custom Sci-fi Tony Stark inspired Desktop operating system It has calculator,timer,task And terminal The front end was made by html,css And javascript

Installation & Development

Method 1: Quick Install (Recommended) If you just want to run the OS on your Windows PC without touching any code, use this method.

Go to the Releases section on the right side of this GitHub page.
Download the latest ARC-OS-Windows.zip file under the "Assets" dropdown.
Right-click the downloaded .zip file and select Extract All.
Open the extracted folder and double-click ARC-OS.exe to launch the operating system.(If Windows Smart screen blog set click more info and run anyway) To exit the OS at any time, click the red [X] Exit icon on the desktop, or press Ctrl + Q.
